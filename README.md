# Coding-for-Economists_Assignment_Hlib-Khalapsus
Assignment for Coding for Economists course

This repository contains Python and Stata scripts for the assignment.

- In order to run **Stata/** script, go to Stata folder and run the master file.do.
The master file contains 2 DO-files that clean and save the new version of data, as well as produce the output necessary for analysis.

- In order to run **Python/** script, simply run the Assignment_Hlib Khalapsus.py file. It contains the code that cleans and saves the new version of data, as well as produces summary stat and histogram of variable Price.
